# 🌦️ WeatherSphere Pro

> A modern, responsive, real-time weather dashboard built using **HTML5, CSS3, and Vanilla JavaScript**. WeatherSphere Pro fetches live weather data from the **OpenWeather REST API** and provides current weather conditions, a 5-day forecast, geolocation support, recent search history, and an elegant glassmorphism user interface.

1. Create a free account on OpenWeather.

2. Generate an API key.

3. Replace:

```javascript
const API_KEY = "YOUR_API_KEY";
```

with your own API key inside `script.js`.
